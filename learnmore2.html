Project

Social Good Slack Bot

As a research assisstant, I was tasked with building a slack bot using AI that was able to comprehend user inputs. With the usages of python, and Slack API, I created an interactive slack bot that receives human input about coding/programming questions and will respond with external links to the answer. The process of creating this was using text classification and web scraping to match similar/specific words. I used logistic regression to build an accurate working model that had a 97% accuracy. The end result was a successful live bot that has helped more than 100+ new program learners understand coding easier.
